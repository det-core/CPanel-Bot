 INSTALLATION

```bash
# Clone repository
git clone https://github.com/det-core/CPanel-Bot.git

# Enter directory
cd CPanel-Bot

# Install dependencies
npm install

# Configure settings
nano settings.js

# Start bot
npm start
```
